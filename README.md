## Technologies
Angular 6, Javascript, Typescript, CSS, HTML.

## Patterns
Module, Singleton and MVC.

## Deploy
### Development Mode (needs NodeJS and Angular CLI)
```
ng serve --open
```

### Production Mode
Copy the contents of the folder "dist" to your web server where you want to display it.
